# Loan-Lending-Platorm

  ### FLIPR Hackathon
  Problem statement : 
   You are expected to build a web application where users can ask or give personal loans to other users of the
    application. This platform can also be called as a C2C (Customer to Customer) personal loan lending platform.
  
## Clone Repository 
```sh
git clone --recursive git@github.com:ShisuiMadara/Loan-Lending-Platorm.git
```
